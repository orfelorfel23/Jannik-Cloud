# Quizmaster

Self-hosted quiz application by [nymanjens/quizmaster](https://github.com/nymanjens/quizmaster).

- **Subdomain**: `quiz.orfel.de`
- **Status**: Prepared but **not enabled**. Create a `service.enabled` file to activate.

## Configuration

The quiz is configured via `quiz-config.yml` placed in the volume at:
```
/mnt/Jannik-Cloud-Volume-01/quizmaster/quiz/quiz-config.yml
```

See the [official documentation](https://github.com/nymanjens/quizmaster) for the config format.
