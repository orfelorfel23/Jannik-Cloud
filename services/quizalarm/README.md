# Quizalarm

Hybrid-Quiz-Plattform für Live-Quiz (Kahoot-ähnlich) und Solo-Lernen mit Baserow als Backend.

## Konfiguration

1. `bash generate-env.sh` — generiert `.env` und `.env.age`
2. `.env.age` committen, `.env` niemals committen
3. Tabellen-IDs werden über das Webinterface unter `/admin` verwaltet

## URLs

- Teilnehmer: `https://quizalarm.orfel.de`
- Admin: `https://quizalarm.orfel.de/admin`
- Präsentation: `https://quizalarm.orfel.de/present`
