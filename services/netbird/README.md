# NetBird VPN

Self-hosted NetBird instance integrated with Jannik-Cloud.

## Setup

1. Run `./generate-env.sh` to create `.env`.
2. Deploy via the master script.

## URLs

* Dashboard: `https://vpn.orfel.de`

## Configuration

* **Internal VPN Range**: `10.23.1.0/23`
* **Identity Provider**: Built-in (Local Accounts)
* Data Volume: `/mnt/Jannik-Cloud-Volume-01/netbird`
